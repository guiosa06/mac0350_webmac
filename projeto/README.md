# SNAKIME
Web app que simula o clássico jogo da cobrinha. Projeto WebMAC da disciplina MAC0350.
## Funcionalidades
- Tela com o jogo, interação pelo teclado, marcador de pontos, etc., implementados com JavaScript
- Tela para criar uma conta simples de usuário (hx-post)
- Tela de login
- Tela de ranking de melhores pontuações (com paginação e navegação entre páginas)
- Opção de deletar pontuações (hx-delete)
- Personalização de cores (hx-put)
- Navegação entre as páginas, com modelo SPA (hx-get)
