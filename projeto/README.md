# Snake Game
Web app que simula o clássico jogo da cobrinha. Projeto WebMAC da disciplina MAC0350.
## Funcionalidades
- Tela com o jogo, interação pelo teclado, marcador de pontos, etc.
- Tela para criar um conta simples de usuário
- Tela de ranking de melhores pontuações
- Opção de deletar pontuações
